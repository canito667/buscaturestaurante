# 🍽️ BuscaTuRestaurante

Decisión rápida para el almuerzo en Francia. Dejas de decir «no sé» y acabas
pidiendo siempre lo mismo en Uber Eats.

- 🌍 Datos **abiertos** de OpenStreetMap (Nominatim + Overpass). Sin Google, sin API key.
- 🧭 Índice de Recomendación X/100 transparente (lo que la comunidad recomienda de verdad).
- 🥡 Filtros móviles: solo/acompañado, para llevar, vegetariano/vegano, abierto ahora.
- 🎲 «No sé, decídeme tú»: elige al azar entre las mejores (con humor).
- 🌐 Español / English / Français.
- 📍 Enlace a Google Maps solo para ver la dirección en el móvil.

## Usar en el iPhone
Ábrela en Safari → Compartir → **Añadir a pantalla de inicio**. Queda como
una app: se abre a pantalla completa.

Fuente: OpenStreetMap (datos abiertos). El índice es un proxy honesto de
recomendación de comunidad.
